# JSCKit
![](https://jitpack.io/v/JustinRoom/JSCKit.svg)
### Usage
+ use with gradle:  

1、Add it in your root build.gradle at the end of repositories:
```
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}
```
2、Add the dependency
```
compile 'com.github.JustinRoom:JSCKit:0.0.1'
```
